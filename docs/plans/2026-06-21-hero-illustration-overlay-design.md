# Hero Illustration Overlay Design

## Goal

새 히어로 일러스트의 넓은 상단 여백을 활용해 이름, 날짜, 장소를 배치하고 기존의 어두운 하단 그라데이션을 제거한다.

## Visual Direction

- 히어로 문구 전체를 화면 상단 약 25% 지점에 가운데 정렬한다.
- 검은색 하단 그라데이션 대신 투명한 오버레이를 사용해 일러스트 원본 색을 보존한다.
- 텍스트는 일러스트 선과 조화를 이루는 딥 브릭 레드 계열로 변경한다.
- 밝은 배경 위 가독성을 위해 매우 약한 밝은 그림자만 더한다.
- 이름, 날짜, 장소의 기존 크기와 정보 위계는 유지한다.

## Responsive Behavior

모바일과 데스크톱 모두 비율 기반 상단 배치를 사용해 일러스트 인물 영역과 겹치지 않게 한다. 화면이 낮아져도 문구가 리본 장식 바로 아래의 빈 공간에 머물도록 안전 여백을 둔다.

실제 모바일 Safari에서는 기존 `clamp(170px, 24svh, 220px)` 상단 여백으로 인해 장소 문구가 인물 머리와 겹쳤다. 중앙 문구는 좌우 리본과 충돌하지 않으므로 `clamp(135px, 17svh, 170px)`로 약 30px 위로 이동해 문구 묶음과 인물 사이의 여백을 확보한다.

## Verification

- 실제 히어로 이미지로 페이지를 열어 문구가 상단 빈 공간 안에 위치하는지 확인한다.
- 어두운 오버레이가 사라지고 글씨가 일러스트와 유사한 색조인지 확인한다.
- 모바일 폭과 데스크톱 폭에서 인물 및 상단 리본과 겹치지 않는지 확인한다.
